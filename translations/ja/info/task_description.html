真の旅人は皆、３つの術を知っていなければならない。すなわち、火を起こすこと、水を見つけること、そして、周辺の自然環境から有益な情報を得ることである。プログラムは、火をつけたり、水をみつける助けにはなりそうにない。しかし、自然環境からの情報抽出に関しては、プログラミングがまさに打って付けなのである。

あなたの任務は、与えられた時刻から、地平線からの太陽の角度を見つけ出す事です。太陽は、朝の6:00に東の地平線から昇りますが、この時の角度を0度とします。そして、太陽はお昼の12:00に天頂に到達します。この時の角度は90度となります。やがて、太陽は夕方の18:00に西の空へ沈みます。この時の角度は180度となります。もし太陽が見えない夜の時刻が与えられた場合（朝の6:00より前や夕方の18:00より後）、あなたの関数は "I don't see the sun!"（太陽は見えないよ！）という文字列を返すようにして下さい。

Input: 時刻

Output: 太陽の角度, 小数点第２位までに丸める.
