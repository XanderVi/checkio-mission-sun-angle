本当の旅人は、３つのことを知っておかなければなりません。火のおこし方、水の見つけ方、自分を取り巻く環境から有用な情報を抽出する方法の３つです。火と水については、プログラミングはあなたの助けにならないでしょう。しかし、情報の抽出となると、まさにあなたの必要としているものに違いありません。

時刻を知るのに使われる、水平線と太陽の角度を調べることが今回のタスクです。入力：太陽は午前６時に東から昇ります。これは角度０度に対応しています。午後１２時に太陽は天頂に達します。これは９０度を意味します。午後６時は太陽が沈む時刻です。従って、角度は１８０度です。もし入力が夜の時間（午前６時より前、もしくは午後６時よりも後）だったら、あなたの作る関数は "I don't see the sun!" と返すべきです。

入力: 時刻

出力: ２桁の十進数に丸めた太陽の角度
